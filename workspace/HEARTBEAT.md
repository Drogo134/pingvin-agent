# HEARTBEAT — Проактивный мониторинг агента

Этот файл выполняется автоматически по расписанию.

## Задачи heartbeat

### Каждые 30 минут (cron: `*/30 * * * *`)
- Проверить входящую почту: `bash /app/workspace/scripts/email_check.sh` → обработать новые письма как EmailIntakeSkill
- Проверить очередь follow-up: применить FollowUpSkill для лидов с меткой follow_up_due
- Проверить handoff-ы: если менеджер не ответил >2 ч в рабочее время → `bash workspace/scripts/notify_managers.sh "⚠️ ..."`

### Каждый час (cron: `0 * * * *`)
- Проверить доступность СБИС: `bash /app/workspace/scripts/sbis_crm.sh test_auth`
- Собрать сводку активных сессий из `sessions_list`

### В 9:00 МСК рабочие дни (cron: `0 6 * * 1-5`)
- Прочитать `logs/analytics_$(date +%Y-%m).jsonl` за вчера
- Отправить утреннюю сводку всем менеджерам: `bash workspace/scripts/notify_managers.sh` (формат ниже)

### В 18:00 МСК рабочие дни (cron: `0 15 * * 1-5`)
- Прочитать аналитику за сегодня
- Отправить итоговую сводку дня всем менеджерам через `notify_managers.sh`

### При команде `/stats` от менеджера
- Применить AnalyticsSkill: посчитать метрики за текущий месяц

## Формат утренней сводки менеджеру

```
📊 Утренняя сводка — [дата]

🆕 Новых обращений за ночь: N
⏳ Ожидают менеджера: N (список с именами)
📅 Follow-up запланировано: N касаний
📨 Новых писем: N (если есть)

📈 За вчера:
  • Диалогов: N | Лидов: N (X%)
  • Полный цикл: N (X%) [цель ≥70%]
  • Handoff: N (X%)

Хорошего рабочего дня! 🐧
```

## Формат итоговой сводки (18:00)

```
📊 Итоги дня — [дата]

💬 Диалогов: N
✅ Лидов создано: N (X%)
🤝 Полный цикл: N (X%)
📋 Передано менеджерам: N

⏳ Открытые без закрытия: N
  — [перечислить имена если есть]

До завтра! 🐧
```
